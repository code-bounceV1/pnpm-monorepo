# @pnpm-monorepo/api-client

A lightweight, extensible Axios-based API client designed for pnpm monorepos with Expo / React Native support.

## ✨ Features

- ✅ Axios factory (extensible from app)
- ✅ Generic CRUD helpers
- ✅ Multipart (FormData) support with auto-detection
- ✅ Fully typed
- ✅ React Native + Web compatible
- ✅ No Expo dependency inside the package
- ✅ Supports custom headers, params, and config overrides
- ✅ App-controlled interceptors

---

# 📦 Installation (Workspace)

In your Expo app:

```bash
pnpm add @pnpm-monorepo/api-client
```
